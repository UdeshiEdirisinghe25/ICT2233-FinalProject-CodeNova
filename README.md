Savory Delights - Food Delivery Web Application
Next.js | Neon PostgreSQL | Vercel Deployment

This repository is created to support database connection(Neon PostgreSql) and deployment configuration for the Savory Delights food delivery system.
The full source code for the coursework is maintained in a private repositary as required.

Project Overview
Savory Delights is a modern food-ordering web application designed to provide with an easy and convenient way to browse food items, place orders, and contact the restaurant.

The main objectives of the system are:

1. Improve customer experience through a responsive, user- friendly interface
2. Allow administrators to manage menu items and track orders efficiently
3. Streamline the food ordering process with a stuctured workflow
4. Maintain system reliability using proper database integration

This public repo acts as the deployment support layer, especially for the database and hosting environment.

Key Features 

Customer Features:
Browse menu items by category
View detailed foof descriptions
Ass items to cart
Place orders online
Contact the restaurant
Fully responsive and user-friendly interface

Admin Features:
Admin login authentication
Add, edit, update and delete menu items
View and manage all customer orders
Organaized dashboard for bette visibility

System Architecture
The system follows a full-stack architecture using Next.js for both frontend UI and backend Api routes.

1. Customer & Admin : Where users and admin interact with the app.
2. Next.js Frontend: Manages the UI and navigation.
3. API Routes: Handle server-side logic and data requests.
4. Neon PostgreSQL Database: Stores all app data like users and tasks.

Core Database Tables

1. Admin - stores administrator login credentials
2. Menu-Items - holds food item details
3. Orders - records customer orders
4. Order_Items - maps order with purchased items

Database (Neon PostgreSQL)

This repository is publicily available only to create and manages:

Database connection
Enviroment configuration for Vercel 
Secure intergration between the app and the Neon PostgresSQL instance

Deployment 
This public repository is linked to Vercel and Vercel connects to the Neon PostgreSQL databse using environment variables.

It ensures:
1. Automatic deployments
2. Stable connection to the cloud databse
3. Envionment isolation for development and production
